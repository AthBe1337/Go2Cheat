# Go2Cheat

CS2内核外挂

通过内核驱动实现内存读写，可以绕过VAC以及一部分平台的反作弊程序。

## 功能

***1.ESP*** 骨骼/方框/血量/护甲/武器/拆弹工具/C4状态

***2.TriggerBot*** 自动扳机

***3.RadarHack*** 雷达透视
